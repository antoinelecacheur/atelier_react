Installation

  • Via votre terminal, dans le répertoire de votre choix :
    npx create-react-app mon-appli
  • Installez Visual Studio Code (pas de besoin de droits d'admin)
  • Ajoutez l'extension React Dev Tools sur votre navigateur (l'extension existe sous chrome et firefox)

Un peu de Javascript avant de se lancer

  • Nouveautés ECMASCRIPT

Structure d'un projet React

Les commandes npm

# ajouter un module npm install nomDuModule # installer tous les modules nécessaires # après avoir récupéré le code source depuis un dépôt git npm install # lancer l'application npm start # créer un livrable npm run build # lancer les tests npm test
