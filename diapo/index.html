Installation

  • Via votre terminal, dans le répertoire de votre choix :
    npx create-react-app mon-appli
  • Installez Visual Studio Code (pas de besoin de droits d'admin)
  • Ajoutez l'extension React Dev Tools sur votre navigateur (l'extension existe sous chrome et firefox)

Un peu de Javascript avant de se lancer

  • Nouveautés ECMASCRIPT

Structure d'un projet React

  • node_modules contient toutes les dépendances du projet. ~ maven repository propre à chaque application React.
  • package.json déclare toutes les dépendances présentes dans l'application.
  • public contient le seul fichier html de l'appli.
  • src qui contiendra tous nos composants.

Les commandes npm

# ajouter un module npm install nomDuModule # installer tous les modules nécessaires # après avoir récupéré le code source depuis un dépôt git npm install # lancer l'application npm start # créer un livrable npm run build # lancer les tests npm test

Passer de l'information entre composants : props

  • React prévoit donc un moyen de communiquer : Composant Parent -> Composant Enfant par l'intermédiaire des props.
  • Les props sont des propriétés qu'un composant parent va passer à un composant enfant.
  • Il est impossible de modifier une props directement dans le composant enfant qui la reçoit.c.
  • Le rôle des props est d'imposer un comportement spécifique au composant en question.
